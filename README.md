# Ice Cream Cone Business Website

This project is a website for a business that manufactures ice cream cones and prints cone sleeves. The website serves as an online presence for the business, providing information about the products and services offered.

## Project Structure

- **src/**
  - **index.html**: Main entry point of the website with navigation links.
  - **about.html**: Information about the business, its history, and mission.
  - **products.html**: Showcases the products offered, including ice cream cones and cone sleeves.
  - **contact.html**: Contains a contact form for customer inquiries.
  - **css/**
    - **styles.css**: Styles for the website, defining layout and appearance.
  
- **images/**
  - **products/**: Directory holding images of the products.

- **js/**
  - **main.js**: JavaScript code for functionality and interactivity.

- **package.json**: Configuration file for npm, listing dependencies and scripts.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies using npm:
   ```
   npm install
   ```
4. Open `src/index.html` in a web browser to view the website.

## Features

- Responsive design for optimal viewing on various devices.
- Easy navigation between different sections of the website.
- Contact form for customer inquiries.
- Showcase of products with images and descriptions.
